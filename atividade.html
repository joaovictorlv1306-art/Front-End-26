Laços de Repetição

O que são?

Laços de repetição são estruturas usadas para repetir um código várias vezes. Eles ajudam a evitar escrever a mesma coisa muitas vezes.

For

Usamos quando sabemos quantas vezes vai repetir.

for (int i = 0; i < 5; i++) {
    System.out.println(i);
}

While

Repete enquanto a condição for verdadeira.

int i = 0;

while (i < 5) {
    System.out.println(i);
    i++;
}

Do While

Executa pelo menos uma vez antes de verificar a condição.

int i = 0;

do {
    System.out.println(i);
    i++;
} while (i < 5);
